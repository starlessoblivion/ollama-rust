# Ollama-Rust Mobile

A lightweight Rust-based web interface for Ollama, optimized for desktop and mobile browsers.

The curl commands handle all dependencies, environment variables, and build the project for you.

## Quick Start

Run the automated installer to handle dependencies and build the project:

```bash
curl -sSL https://raw.githubusercontent.com/starlessoblivion/ollama-rust/main/install.sh | bash
